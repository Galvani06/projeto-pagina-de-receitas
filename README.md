🥘 Página de Receita - Strogonoff de Frango
Este projeto é uma página web estática desenvolvida durante o curso da Rocketseat, focada na estruturação semântica de conteúdo e na aplicação de boas práticas de segurança e acessibilidade.

🛡️ Foco em Cybersecurity: Implementação de CSP
Diferente de uma implementação padrão, este projeto integra uma camada de defesa ativa utilizando Content Security Policy (CSP) via meta tag.

Finalidade da CSP neste projeto
A CSP atua como uma barreira de segurança que define quais fontes de conteúdo (estilos, imagens e fontes) o navegador tem permissão para carregar.

Prevenção de ataques XSS (Cross-Site Scripting): Bloqueia a execução de scripts maliciosos injetados por terceiros.

Controle de Recursos Externos: Garante que apenas fontes e estilos vindos de domínios confiáveis, como o Google Fonts, sejam renderizados.

Segurança de Dados: Restringe a origem de objetos e frames, diminuindo a superfície de ataque da aplicação.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica completa (main, section, footer).

CSS3: Estilização responsiva e ajustes de layout (flexbox/grid).

Segurança Web: Implementação de CSP e atributos rel="noopener noreferrer" em links externos.
