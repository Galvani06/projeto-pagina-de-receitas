🚀 Tecnologias
Este projeto foi desenvolvido utilizando as seguintes tecnologias e conceitos:
HTML5: Estruturação semântica para melhor acessibilidade e SEO.
CSS3: Estilização responsiva e layouts modernos.
Cibersegurança (Diferencial): Implementação de Política de Segurança de Conteúdo (CSP).
Git e GitHub: Controle de versão e documentação profissional.
Figma: Prototipagem e guia visual do projeto.

💻 Projeto
Este projeto foi desenvolvido como parte da formação da Rocketseat para consolidar os fundamentos de HTML e CSS. A aplicação consiste em uma página de receitas focada em clareza visual e estruturação de dados.

🛡️ Prática de Cibersegurança Implementada (Diferencial)
Embora o foco inicial do curso fosse o desenvolvimento web básico, decidi aplicar práticas de cibersegurança de forma independente para alinhar o projeto aos meus objetivos profissionais na área de Segurança da Informação.

Implementação de CSP: Adicionei uma camada de defesa ativa via meta tag para definir quais fontes de conteúdo (estilos e fontes) o navegador tem permissão para carregar.

Prevenção de Ataques: A finalidade principal é mitigar vulnerabilidades de XSS (Cross-Site Scripting) e injeções de dados maliciosos.

Controle de Recursos Externos: Configurei a política para garantir que apenas recursos vindos de domínios confiáveis, como o Google Fonts, sejam renderizados com segurança.
